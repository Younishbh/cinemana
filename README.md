# 🎬 سينمانا - منصة الأفلام والمسلسلات

منصة ويب حديثة ومتقدمة متخصصة في عرض وتصفح الأفلام والمسلسلات العالمية والعربية.

## ✨ المميزات

- 🎞️ **عرض الأفلام**: مكتبة كبيرة من الأفلام المختلفة
- 📺 **عرض المسلسلات**: تصفح جميع المسلسلات الشهيرة
- 🔥 **الأكثر مشاهدة**: قسم خاص بأكثر المحتويات مشاهدة
- 🔍 **بحث متقدم**: بحث سريع عن الأفلام والمسلسلات
- 📱 **تصميم متجاوب**: يعمل على جميع الأجهزة (ديسكتوب، تابلت، موبايل)
- ⭐ **التقييمات**: عرض تقييمات المحتوى
- 📋 **تفاصيل كاملة**: معلومات شاملة عن كل محتوى
- 🎨 **تصميم عصري**: واجهة مستخدم جميلة وسهلة الاستخدام

## 🚀 التكنولوجيا المستخدمة

- **HTML5**: للهيكل الأساسي
- **CSS3**: للتصميم والرسوميات
- **JavaScript (Vanilla)**: للوظائف والتفاعلات
- **Responsive Design**: تصميم متجاوب

## 💻 طريقة الاستخدام

### التثبيت
```bash
# استنساخ المستودع
git clone https://github.com/Younishbh/cinemana.git

# الدخول إلى المجلد
cd cinemana

# فتح الملف index.html في المتصفح
```

### الميزات الأساسية

1. **التصفح**: تصفح الأفلام والمسلسلات بسهولة
2. **الفلترة**: فلترة المحتوى (الكل - أفلام - مسلسلات - أكثر مشاهدة)
3. **البحث**: البحث السريع عن محتوى معين
4. **التفاصيل**: عرض تفاصيل كاملة عند الضغط على أي محتوى
